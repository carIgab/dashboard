# dashboard
- Descrição de como fazer dashboard

- Qual objetivo de dashboard? quais informações (dados trabalhados,capaz de fornecer um tomada decisão, planejamento e organização) são importantes e cruciais para cumprir o objetivo
- Trabalhar no base de dados: tirar dados irrelevantes, duplicatos ou invalidos, formatar corretamente com dados, para organização, tirar separados os tabelas e calculos que serão utilizados big number e para graficos
- Deixar os dashboard intuitivo, simples (sem muitos gráficos, querendo mostrar tudo em dashboard, deixando muito poluido, mostrar apenas informações relevantes), com aparência limpa (sem o carnaval de cores)
